# MongoDB MCP
Repository for testing server customizations and integrations with MCP. Exploring features and configurations to optimize functionality

## Getting Started
```sh
netstat -ano | find ":8008"

tasklist | find "node.exe"

taskkill /IM node.exe /F
```
